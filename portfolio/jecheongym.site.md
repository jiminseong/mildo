- 반응형 웹 기반으로 국문/영문 다국어 페이지를 구현하고 SEO 기본 구조를 함께 설계했습니다.

3) 주요 업무
- 히어로, 운영 철학, 이용 규칙, 장비 라인업, 위치/문의 섹션 등 랜딩 페이지 전체 구조를 설계하고 구현했습니다.
- 57종 장비를 9개 브랜드와 부위 기준으로 탐색할 수 있는 장비 소개 UI와 데이터 구조를 구성했습니다.
- 카카오톡 ID 복사, 주소 복사, 카카오맵/네이버지도 이동 기능을 연결해 문의 전환 동선을 단순화했습니다.

4) 주안점
- 단순 소개형 홈페이지가 아니라 센터의 운영 철학, 규칙, 장비 경쟁력을 한 화면 안에서 명확하게 전달하는 데 집중했습니다.
- Open Graph, robots.txt, sitemap.xml, ExerciseGym 구조화 데이터, 네이버 사이트 검증 메타 태그를 적용해 검색 친화 구조를 마련했습니다.
- 제조사 공식 이미지를 활용해 신뢰도 높은 콘텐츠를 구성하고, 향후 장비 추가가 쉬운 정적 데이터 구조로 설계했습니다.

## 3. 프로젝트 배경

### 1) 문제점
- 오프라인 기반 헬스장의 전문성과 실제 보유 장비 수준이 온라인에서 충분히 전달되지 않았습니다.
- 일반적인 피트니스 센터와 다른 운영 철학과 이용 규칙을 사전에 명확히 전달할 필요가 있었습니다.
- 장비 정보, 위치 안내, 문의 방법이 분산되어 있어 모바일 사용자의 탐색 및 문의 전환 흐름이 길었습니다.

### 2) 프로젝트 목표
- 제천 웨이트 전문 헬스장의 정체성을 선명하게 보여주는 공식 랜딩 페이지를 구축합니다.
- 장비 라인업, 운영 규칙, 위치 및 문의 정보를 한 페이지 안에서 빠르게 확인할 수 있도록 구성합니다.
- 카카오톡 문의, 주소 복사, 지도 이동까지 이어지는 전환 동선을 직관적으로 설계합니다.
- 국문/영문 페이지와 기본 SEO 세팅을 적용해 검색 유입 기반을 확보합니다.

### 3) 주안점
- AI 느낌 없이 실제 시설과 브랜드 이미지를 중심으로 신뢰도 높은 화면을 구성했습니다.
- 제조사 공식 이미지를 기반으로 장비 소개 콘텐츠를 정리해 정보 정확도를 높였습니다.
- 유지보수가 쉬운 정적 데이터 구조를 사용해 신규 장비 추가와 향후 확장에 대응할 수 있게 설계했습니다.

## 7. 포트폴리오 이미지 등록 가이드

### 업로드 순서 (총 10장)

#### 이미지 1
- 캡처 화면: 메인 히어로 섹션
- 설명: 강한 브랜드 카피와 CTA를 전면에 배치해 제천 웨이트 전문 헬스장의 첫인상을 전달한 메인 화면.

#### 이미지 2
- 캡처 화면: Identity 섹션
- 설명: 센터의 운영 철학과 타깃 사용자 기준을 텍스트 중심으로 명확히 전달한 브랜드 소개 섹션.

#### 이미지 3
- 캡처 화면: Message from Gym 섹션
- 설명: 이용 규칙과 운영 메시지를 강조형 레이아웃으로 구성해 사전 필터링 역할을 하도록 설계한 화면.

#### 이미지 4
- 캡처 화면: 브랜드 로고 마퀴 + 장비 라인업 + 필터 화면
- 설명: 브랜드 로고, 장비 라인업, 브랜드·부위 필터를 한 화면에 담아 장비 규모와 탐색 기능을 함께 보여준 화면.

#### 이미지 5
- 캡처 화면: 신규 장비 카드 강조 화면
- 설명: 신규 입고 장비 카드를 전면에 배치해 실운영 기반 업데이트 구조와 장비 경쟁력을 보여준 화면.

#### 이미지 6
- 캡처 화면: 장비 카드 리스트 상세 화면
- 설명: 장비 카드의 브랜드, 부위, 상태 정보를 구조화해 콘텐츠 완성도와 데이터 정리 수준을 보여준 화면.

#### 이미지 7
- 캡처 화면: 문의 CTA 강조 화면
- 설명: 메인 CTA와 문의 유도 흐름을 명확히 배치해 사용자의 행동 전환을 고려한 랜딩 구조를 보여준 화면.

#### 이미지 8
- 캡처 화면: LOCATION & INFO 섹션
- 설명: 주소, 운영 시간, 카카오톡 ID, 지도 이동 버튼을 통합해 문의 전환 흐름을 단순화한 화면.

#### 이미지 9
- 캡처 화면: 영문 페이지 또는 언어 전환 화면
- 설명: 국문·영문 로케일 전환을 지원해 다양한 사용자가 동일한 정보 구조로 접근할 수 있게 한 화면.

#### 이미지 10
- 캡처 화면: 모바일 반응형 화면
- 설명: 모바일 환경에서도 핵심 정보와 CTA가 자연스럽게 이어지도록 반응형으로 최적화한 화면.

### 캡처 팁
- 데스크톱 8장, 모바일 2장 정도 비율로 구성하면 가장 안정적입니다.
- 장비 섹션은 통합 화면 1장과 카드/CTA 강조 화면 2~3장으로 구성하면 중복이 줄어듭니다.
- 브라우저 전체보다 서비스 영역 위주로 잘라 여백을 줄이는 편이 더 깔끔합니다.
