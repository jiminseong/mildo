## 2. 프로젝트 배경

### 복붙용 문안

```text
1) 문제점
- 카페24 기본 구조만으로는 상품 상세 이미지 확대 경험이 제한적이어서 제품 디테일을 충분히 보여주기 어려웠습니다.
- 메인 페이지의 버튼 위치, 배너 텍스트, 아이콘 크기와 정렬이 화면별로 일관되지 않아 브랜드 완성도가 떨어질 수 있었습니다.
- 국문/영문 메인 섹션이 분리 운영되는 구조에서 카피와 레이아웃을 함께 정비할 필요가 있었습니다.

2) 프로젝트 목표
- 상품 상세 페이지에 모바일과 데스크탑 모두 대응하는 커스텀 줌 기능을 적용합니다.
- 메인 페이지의 CTA 위치, 배너 텍스트, 그리드 배치, 아이콘 비율을 정리해 시각 완성도를 높입니다.
- 국문/영문 메인 화면과 관리용 소스를 함께 정비해 브랜드 메시지를 일관되게 전달합니다.

3) 주안점
- 카페24 기존 모듈 구조를 깨지 않으면서 커스텀 인터랙션을 구현
- 모바일 터치 동작과 데스크탑 마우스 동작을 분리 설계
- 실서비스 반영을 전제로 작은 UI 디테일까지 반복 조정
```

## 3. 프로젝트 상세

### 복붙용 문안

```text
1) 프로젝트 소개
- MOUMOUTE 공식몰 퍼블리싱 외주는 카페24 기반 주얼리 브랜드 이커머스 사이트의 상품 상세 경험과 메인 페이지 완성도를 높이기 위한 작업입니다.
- 단순 스타일 수정이 아니라 상품 상세 인터랙션, 아이콘 SVG, 메인 비주얼 레이아웃, 국문/영문 메인 섹션까지 실서비스 화면 전반을 조정했습니다.

2) 담당 범위
- 기존 카페24 템플릿 구조를 분석하고 상품 상세용 zoom.css, zoom.js를 추가해 커스텀 확대 기능을 구현했습니다.
- 글로벌/장바구니 아이콘 크기 및 위치를 수정하고, 메인 배너 버튼과 텍스트 위치를 여러 시안으로 조정한 뒤 확정안을 반영했습니다.
- 국문 main_control.html, 영문 main01.html, customer.js, main.css 계열 소스를 수정해 메인 섹션 카피와 레이아웃을 정리했습니다.

3) 상품 상세 인터랙션 구현
- 상품 상세 영역 이미지 선택자를 기준으로 확대 대상 이미지를 잡고, 원형 렌즈 형태의 줌 UI를 커스텀 구현했습니다.
- 데스크탑에서는 클릭 후 마우스 이동으로 렌즈가 따라오도록 설계했고, 모바일에서는 확대 토글 버튼을 두어 터치 스크롤과 충돌하지 않도록 처리했습니다.
- 이미지 소스는 ec-data-src와 기본 src를 함께 대응하도록 구성해 실제 운영 이미지 구조에서도 안정적으로 동작하게 만들었습니다.

4) 메인 페이지 UI 개선
- 메인 3섹션의 버튼 위치와 배너 카피 배치를 여러 안으로 검토한 뒤 최종안으로 정리했습니다.
- half_section, one_section, main03, main04 등 메인 레이아웃의 그리드와 반응형 규칙을 조정해 데스크탑과 모바일 모두에서 시각 흐름이 자연스럽도록 만들었습니다.
- New arrivals, Home 등의 CTA 문구와 국문/영문 카피 블록을 실제 브랜드 톤에 맞춰 반영했습니다.

5) 운영 및 반영 포인트
- 카페24 detail.html, main.html 계열 템플릿에 직접 연결되는 형태로 CSS와 JavaScript를 추가해 별도 플러그인 없이 적용했습니다.
- 모바일 1안, 2안, 3안과 데스크탑 시안을 비교하며 기능성과 화면 균형을 함께 검토했고, 확정안 기준으로 실서비스 반영 방향을 정리했습니다.
- 단순 시안 제안이 아니라 실제 적용 가능한 코드와 관리용 소스 변경까지 포함한 퍼블리싱 작업으로 마무리했습니다.
```
