# 숨고 프로필 입력본 (최종)

## 1) 숨고 활동명 (SEO 고려)

3일데모 웹개발 홈페이지제작 관리자페이지

## 2) 한 줄 소개 (수정본)

말보다 데모로 증명합니다. 3일 안에 방향 확인용 데모를 만들고, 운영 가능한 웹서비스로 완성합니다.

## 2-1) 후킹 이미지 문구 (배너/소개 이미지용)

- 3일 안에, "말"이 아니라 "데모"를 보여드립니다
- 기획-디자인-개발-배포까지 끊김 없이 진행
- 빠른 제작 + 실제 운영 기준까지 함께 설계

## 3) 서비스 상세 설명

안녕하세요. 웹 개발/홈페이지 제작/관리자 페이지 구축을 중심으로 작업하는 개발자입니다.

단순 화면 제작이 아니라, "실제로 운영 가능한 결과물"을 만드는 것을 기준으로 진행합니다.
프로젝트마다 기획 의도를 먼저 정리한 뒤,
1) 화면/사용자 흐름 설계
2) 프런트엔드 구현
3) 백엔드/API 및 데이터 구조 연결
4) 배포 및 운영 기준 정리
순서로 작업합니다.

특히 아래 유형에 강점이 있습니다.
- 소개형/브랜드형 홈페이지 제작
- 예약/문의/신청 흐름이 있는 서비스형 웹
- 관리자 페이지(로그 조회, 상태 관리, 운영 기능)
- 데모/프로토타입을 빠르게 만들어 의사결정 속도를 높이는 방식

진행 중에는 중간 결과를 자주 공유해 방향 이탈을 줄이고,
완료 후에도 실제 운영에서 필요한 수정 포인트를 빠르게 반영합니다.

## 4) 질문답변 입력본

### Q1. 서비스가 시작되기 전 어떤 절차로 진행하나요?
문의 내용을 확인한 뒤 요구사항을 구조화해 1차 범위(기능/일정/예산)를 정리합니다.
이후 화면 흐름 또는 와이어프레임 기준으로 합의하고, 개발-중간공유-수정-검수-배포 순서로 진행합니다.
프로젝트 성격에 따라 데모 버전을 먼저 제공해 방향을 빠르게 맞춥니다.

### Q2. 어떤 서비스를 전문적으로 제공하나요?
홈페이지 제작, 랜딩페이지 제작, 웹서비스 개발, 관리자 페이지 구축을 전문적으로 제공합니다.
React/Next.js 기반 프런트엔드와 API/DB 연동, 배포까지 일괄 대응이 가능합니다.
기획 단계가 완성되지 않은 상태에서도 요구사항을 정리해 구현 가능한 형태로 전환해드립니다.

### Q3. 서비스의 견적은 어떤 방식으로 산정 되나요?
페이지 수가 아니라 기능 단위(화면 난이도, 데이터 구조, 관리자 기능, 외부 연동 여부)로 산정합니다.
같은 홈페이지라도 문의/예약/결제/운영 기능 포함 여부에 따라 견적이 달라집니다.
초기 미팅 후 필수 범위와 확장 범위를 나눠, 우선 개발 범위를 명확히 제안드립니다.

### Q4. 완료한 서비스 중 대표적인 서비스는 무엇인가요? 소요 시간은 얼마나 소요 되었나요?
대표적으로 브랜드 홈페이지, 체육시설 웹사이트, 앱 소개/운영 페이지, 연구용 프로토타입 웹을 진행했습니다.
일반 소개형 사이트는 보통 1~3주, 기능형 웹서비스/관리자 페이지 포함 프로젝트는 3~8주 내외가 많았습니다.
일정은 요구사항 확정 속도와 피드백 주기에 따라 가장 크게 달라집니다.

### Q5. A/S 또는 환불 규정은 어떻게 되나요?
완료 직후 발견되는 버그/오동작은 합의된 범위 내에서 무상 수정합니다.
기획 변경이나 신규 기능 추가는 별도 범위로 분리해 일정/비용을 안내드립니다.
착수 전 합의 문서 기준으로 진행하므로, 작업 범위와 수정 기준을 명확하게 관리합니다.
