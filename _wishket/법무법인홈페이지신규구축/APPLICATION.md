# 법무법인 홈페이지 신규 구축 지원서 초안

## 기본 입력값

- 지원 금액: 4,000,000원
- 지원 기간: 30일
- 관련 포트폴리오: 있습니다

### 금액 조정 사유(간단)

- 선제 데모 제작으로 기획/시안 검증 구간의 초기 리스크와 커뮤니케이션 비용을 이미 줄였기 때문에, 품질 기준은 유지한 채 1차 구축 기준으로 400만원 제안이 가능합니다.

## 제출용 본문

```text
<프로젝트 진행 제안>
[프로젝트 이해]
- 이번 공고는 성범죄, 마약 분야에 특화된 법무법인 신규 홈페이지를 구축하면서, 단순 소개 사이트가 아니라 신뢰 형성과 상담 전환이 함께 중요한 프로젝트로 이해했습니다.
- 핵심은 법무법인 소개, 변호사 소개, 연혁, 오시는 길 같은 기본 정보 페이지를 정리하는 것에 더해, 전문분야 상세 페이지, 성공사례 아카이브, 온라인 상담 신청, CMS 운영 구조를 하나의 흐름으로 설계하는 데 있다고 봤습니다.
- 법률 서비스 홈페이지는 첫 화면 신뢰 형성 요소, 전문성 있는 콘텐츠 구조, 상담 CTA 배치, 성공사례 익명화 기준이 문의 전환에 직접 연결되기 때문에 초기 IA와 콘텐츠 분류 기준을 먼저 잡는 것이 중요합니다.

[구축 범위 제안]
- 현재 예산과 기간 기준에서는 1차 구축 범위를 법무법인 소개, 전문분야 상세, 성공사례 관리, 온라인 상담 신청, 기본 CMS, 반응형 웹, 서버/DB/배포 구성까지로 명확히 잡는 것이 적절합니다.
- 상담 후 CRM 연동, 광고 랜딩 세분화, 고급 통계 대시보드, 다국어, 대량 칼럼형 콘텐츠 운영 기능은 2차 고도화로 분리하는 편이 안정적입니다.
- 성공사례는 카테고리, 결과, 사건 키워드, 등록일, 공개 여부 기준으로 관리자가 쉽게 등록, 수정, 비노출 처리할 수 있도록 설계하겠습니다.

[기술 제안]
- 프론트엔드는 Next.js + TypeScript + Tailwind CSS 기반으로 구성해 SEO와 반응형 품질을 확보하고, 콘텐츠 중심 페이지에서도 빠른 로딩과 확장성을 함께 가져가겠습니다.
- 백엔드와 CMS는 Supabase(PostgreSQL/Storage) 기반으로 제안드리며, 성공사례, 전문분야, 변호사 소개, 상담 신청 데이터를 한 관리자 화면에서 일관되게 관리할 수 있도록 구성하겠습니다.
- 검색엔진 유입을 고려해 메타데이터, 오픈그래프, 구조화된 정보, 기본 분석 도구 삽입도 함께 정리할 수 있습니다.

[예상 일정]
- 1주차: 요구사항 정리, 기존 사이트/콘텐츠 분석, IA 및 화면 구조 정의
- 2주차: 와이어프레임 및 메인 UI/UX 디자인
- 3주차: 소개/전문분야/성공사례/상담 페이지 프런트엔드 구현
- 4주차: CMS, 상담 접수 저장, 반응형 보정, QA, 배포

[착수 전 확인 사항]
- 기존 홈페이지 원본 소스 또는 관리자 접근 가능 여부
- 이관 대상 콘텐츠 범위와 문구 확정 여부
- 성공사례 익명화 기준과 공개 승인 절차
- 상담 신청 접수 후 알림 방식(이메일, 문자, 관리자 확인 등)
- 서버/도메인/호스팅 운영 기준

[공유 및 산출물]
- 진행 중에는 주요 화면 단위로 데모를 공유하고, 기획 문서와 화면 설계 기준을 함께 정리해 수정 비용이 커지기 전에 방향을 맞추겠습니다.
- 산출물은 IA/화면 설계, 디자인 원본, 개발 소스, 관리자 기능, 배포 기준 문서까지 정리 가능합니다.

<유사 프로젝트 진행 경험>
[반응형 브랜드/홈페이지 구축 경험]
- 제천남여헬스클럽, 새영동숯불갈비 프로젝트에서는 기획, 디자인, 개발을 일괄로 진행하며 브랜드 소개, 위치/문의 동선, 반응형 웹 구조를 실제 운영 가능한 형태로 구축했습니다.
- 이번 프로젝트에서도 법무법인 소개, 전문분야, 문의 CTA, 오시는 길 등 신뢰 형성과 전환이 함께 필요한 구조를 같은 기준으로 정리할 수 있습니다.

[운영자 관리 기능 경험]
- 비공개 포토부스 운영 프로젝트에서는 운영자 화면과 설정 관리 구조를 직접 설계, 구축했고, 콘텐츠 및 설정 변경이 실제 운영 흐름에 바로 반영되는 구조를 만들었습니다.
- 이 경험을 바탕으로 성공사례 등록, 수정, 숨김 처리와 상담 신청 조회가 가능한 CMS 화면을 실무적으로 설계할 수 있습니다.

[자체 서비스 설계 경험]
- AIM 앱 패밀리를 직접 기획, 설계, 개발, 운영하면서 사용자 흐름과 데이터 구조, 관리자 관점의 운영 기준을 함께 설계해왔습니다.
- 법무법인 홈페이지는 회원 서비스보다 콘텐츠 구조와 상담 전환이 중요하지만, 운영 편의성을 고려한 데이터 모델과 상태 관리 경험은 그대로 적용 가능합니다.

[이번 프로젝트 적용 포인트]
- 이번 공고에서는 법률 전문성을 많은 문구보다 정리된 정보 구조로 보여주는 것이 중요하다고 봅니다.
- 따라서 첫 화면에서는 신뢰 형성 요소를, 전문분야 상세에서는 절차/대응 포인트/FAQ를, 성공사례에서는 탐색 가능한 아카이브를, 관리자 화면에서는 빠른 운영성을 중심으로 설계하겠습니다.

[기술 스택]
```
