# 법무법인 홈페이지 DESIGN SYSTEM

## 1. 디자인 원칙

- 한국형 신뢰 구조: 권위 표현보다 사용자 불안을 낮추는 정보 순서(사건 이해 -> 대응 방법 -> 상담 진입)를 우선한다.
- 상담 전환 중심: 페이지마다 최소 1개 이상의 고정 CTA를 배치하고, 긴급 사용자 동선을 짧게 유지한다.
- 전문성 가독성: 장식보다 문장 가독성과 정보 구획(사건 유형, 대응 절차, 결과)을 명확히 드러낸다.
- 법률 업종 톤: 과한 테크 감성보다 단정하고 묵직한 인상을 유지한다.

## 2. 브랜드 톤앤매너

- 키워드: 신뢰, 절제, 명료, 긴급대응
- 카피 톤: 단문 중심, 단정한 존댓말, 모호한 과장 지양
- 메시지 구조:
  - 문제 인지: "현재 상황에서 무엇이 위험한가"
  - 대응 안내: "지금 무엇을 해야 하는가"
  - 행동 유도: "전화/온라인 상담으로 연결"

## 4. 타이포그래피

- Heading: `Noto Serif KR`
- Body/UI: `Noto Sans KR`
- 숫자 강조: Body 폰트 유지, `font-variant-numeric: tabular-nums`

### 타입 스케일

- Hero Title: 40/52, 700
- Section Title: 30/42, 700
- Card Title: 22/32, 700
- Body: 17/30, 400
- Caption: 14/22, 400

## 5. 레이아웃 시스템

- 기준 폭: 1200px 컨테이너
- 반응형 브레이크포인트:
  - Mobile: 360~767
  - Tablet: 768~1023
  - Desktop: 1024+
- 섹션 간격:
  - Desktop: 96px
  - Mobile: 64px

## 6. 컴포넌트 규칙

### 버튼

- Primary CTA: 진한 네이비 배경 + 금색 텍스트/보더
- Secondary CTA: 연한 배경 + 진한 보더
- 텍스트: "전화 상담", "상담예약", "사례 더보기"처럼 동작이 드러나는 명사형

### 카드

- 성공사례 카드: 사건 분야, 결과 태그, 제목, 요약, 담당 변호사 순서
- 분야 카드: 핵심 위험 포인트 2~3개를 먼저 노출

### 신뢰 요소

- 실명/직함 기반 변호사 프로필
- 대표번호, 지역, 상담 가능 시간
- 결과 태그(불송치, 집행유예, 무혐의 등)

## 7. 인터랙션 규칙

- 애니메이션은 200~350ms 범위의 페이드/슬라이드만 사용
- 강한 패럴랙스, 과한 스크롤 이펙트는 사용하지 않음
- CTA hover는 명도 변화와 미세한 이동만 적용

## 8. 콘텐츠 규칙

- 성공사례 제목 템플릿:
  - "[사건유형] 혐의, [핵심 결과] 이끌어낸 사례"
- 분야 설명 템플릿:
  - "어떤 상황인지" -> "위험 포인트" -> "즉시 대응" 순서
- 상담 안내 문장:
  - 2문장 이내, 법률 용어 최소화

## 9. 접근성 및 컴플라이언스

- 모든 입력 폼은 label 제공
- 색상 대비는 WCAG AA 이상 유지
- 개인정보 동의 체크는 제출 필수
- 푸터에 광고책임변호사/사업자/주소/연락처 고지 가능 구조 유지

## 10. 데모 적용 우선순위

1. 메인 Hero + 고정 CTA + 대표번호
2. 성공사례 아카이브 필터/검색
3. 전문분야 상세의 대응 절차/FAQ
4. 온라인 상담 폼
5. CMS 운영 화면(목업)
