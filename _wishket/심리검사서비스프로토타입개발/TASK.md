# 심리검사 서비스 프로토타입 작업 계획 (최종)

기준 문서: OVERVIEW.md

## 1. 목표

- 연구과제 목적에 맞는 웹 심리검사 프로토타입 구축
- 검사 참여 -> 결과 확인 -> PDF 다운로드 -> 로그 조회까지 단일 흐름 구현
- 계산 로직 재현성과 검증 가능성을 최우선으로 관리

## 2. 구현 위치

- 프로젝트 루트: wishket/심리검사서비스프로토타입개발
- 핵심 문서: APPLICATION.md, PRD.md, TASK.md, DESIGNSYSTEM.md
- 로컬 데모 실행 계획: LOCAL_DEMO_TASK.md
- 앱 루트(예정): wishket/심리검사서비스프로토타입개발/psych-assessment-prototype

- 무회원 검사 진입

- 결과 리포트
- PDF 다운로드
- 엑셀 로직 이관
- 관리자 접속 로그
- 서버/DB/배포

### 3.2 2차 고도화
- 고급 통계 대시보드
- 인증/권한 확장
