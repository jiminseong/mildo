# 심리검사 서비스 프로토타입 개발 지원서 (최종)

## 기본 입력값

- 지원 금액: 4,000,000원
- 지원 기간: 30일
- 관련 포트폴리오: 있습니다

### 금액 조정 사유(명확)

- 본 제안 금액 4,000,000원은 공고에 명시된 핵심 범위(무회원 검사, 엑셀 로직 이관, 결과 리포트/PDF, 관리자 로그, 서버/DB/배포)를 모두 포함한 금액입니다.
- 금액을 낮춘 이유는 난이도를 낮게 본 것이 아니라, 선행 분석과 프로토타입 방식으로 초기 시행착오와 커뮤니케이션 비용을 줄여 전체 개발 효율을 높일 수 있기 때문입니다.

## 제출용 본문

```text
<프로젝트 진행 제안>
[프로젝트 이해]
- 본 프로젝트는 연구과제 수행을 위한 웹 기반 심리검사 프로토타입 개발로 이해했습니다.
- 핵심은 회원가입 없이 누구나 검사에 참여할 수 있는 흐름을 만들고, 기존 엑셀 계산 로직(통계 처리 포함)을 웹에서 동일하게 재현하는 것입니다.
- 단순 설문 화면이 아니라, 응답 저장 -> 결과 리포트 즉시 확인 -> PDF 출력 -> 관리자 로그 확인까지 실제 연구 운영에 필요한 흐름을 검증하는 과업입니다.

[구축 범위 제안]
- 1차 구축 범위는 다음으로 제안드립니다.
  1) 무회원 검사 진입 + 순차 응답 UI
  2) 검사 완료 후 즉시 결과 리포트
  3) 결과 PDF 다운로드
  4) 응답 데이터 저장/가공
  5) 엑셀 계산식 및 t-test 처리 로직 이관
  6) 관리자 접속 로그 조회
  7) 서버/DB/배포 구성
- 위 범위는 본 제안 금액(4,000,000원)에 포함된 구현 범위입니다.

[심리전문가 관점 품질 약속]
- 화면 완성도보다 먼저 "엑셀 결과와 웹 결과가 일치하는지"를 검증표로 확인하겠습니다.
- 해석 문구는 척도별 구간 기준을 먼저 합의한 뒤 반영해, 임의 해석이 섞이지 않도록 관리하겠습니다.
- 응답자가 검사 중 이탈하지 않도록 문항 흐름, 진행률, 안내 문구를 단순하고 명확하게 설계하겠습니다.

[기술 제안]
- 프런트엔드: Next.js + TypeScript + Tailwind CSS
- 백엔드/DB: Node 기반 API + PostgreSQL(Supabase 포함 가능)
- 리포트: HTML 기반 결과 리포트 + PDF 다운로드
- 계산 로직은 화면 코드와 분리해, 엑셀 원본과 대조 검증이 가능한 구조로 구현하겠습니다.

[예상 일정]
- 1주차: 요구사항/엑셀 로직 분석, 데이터 모델 확정
- 2주차: 검사 플로우 및 결과 리포트 UI 구현
- 3주차: 계산 모듈 이관, PDF, 관리자 로그 구현
- 4주차: QA, 검증, 배포, 산출물 정리

[착수 전 확인 사항]
- 문항 목록 확정본(문항 ID/척도/역채점 여부)
- 엑셀 계산식 원본(시트 구조/수식/통계 기준)
- 결과 해석 문구 기준(구간별 설명)
- PDF 결과물 요구 형식(출력 항목/고정 레이아웃)
- 운영 도메인 및 배포 방식

[공유 및 산출물]
- 단계별 데모 URL과 엑셀-웹 비교표를 함께 공유해 계산 정확도를 먼저 검증하겠습니다.
- 산출물은 소스코드, 서버/DB 설계 내역서, 기능 명세 및 운영 가이드 기준으로 정리 가능합니다.

<유사 프로젝트 진행 경험>
- 직접적인 심리검사 상용 서비스 경험을 과장하지 않겠습니다.
- 다만 AIM 앱 패밀리 운영 과정에서 데이터 저장 구조, 결과 화면 구성, 운영 관점의 상태 관리를 직접 설계/개발해왔습니다.
- 포토XX 프로젝트에서는 사용자 화면과 관리자 화면을 분리 설계해 운영 모니터링 흐름을 실제로 구축했습니다.
- 이번 프로젝트 역시 계산 로직 재현성과 결과 신뢰도를 최우선으로 접근하겠습니다.

<사용 기술과 툴>
[기술 스택]
- 프론트엔드: Next.js, React, TypeScript, Tailwind CSS
- 백엔드/DB: PostgreSQL, Supabase(선택)
- 리포트: 웹 리포트 + PDF 출력

[커뮤니케이션]
- 위시켓 메시지/온라인 미팅 기준으로 진행하고, 이슈는 화면+문서로 즉시 공유해 의사결정 속도를 높이겠습니다.

관련 포트폴리오:
- AIM 앱 패밀리: https://aim-official.vercel.app
- 제천남여헬스클럽: https://jecheongym.site
- 포토XX(비공개): 운영자 어드민 및 설정 관리 기능 (미팅 시 상세 공유)
```
