[프로젝트 개요]
연구과제 수행을 위한 웹 기반 심리검사 프로토타입 개발

[프로젝트 배경 및 목표]
- 현재 아날로그 및 엑셀 기반으로 제작된 심리검사 도구를 웹 서비스로 구현하는 연구 개발 프로젝트입니다.
- 응답자가 온라인 환경에서 간편하게 검사에 참여하고, 기존 엑셀의 계산 로직을 자동화하여 즉각적으로 분석 리포트를 제공하는 프로토타입을 구축하는 것을 목표로 합니다.
- 본 프로토타입의 성공적인 개발을 통해 검사의 접근성과 효율성을 검증하고, 향후 상용화 서비스로의 발전 가능성을 모색하고자 합니다.

[과업 범위]
1. 수행 범위
- 프런트엔드/Client 개발
- 백엔드 개발
- 데이터 로그 확인 페이지 (관리자)
- 서버/DB/인프라 구성

2. 상세 기능 요구 사항
2-1. 사용자용 검사 기능: 회원가입 절차 없이 누구나 접근하여 검사를 진행할 수 있는 웹 페이지를 개발합니다.
- 심리검사 진행: 주어진 문항에 대해 순차적으로 응답을 입력하는 인터페이스
- 결과 확인: 검사 완료 후, 분석된 결과 리포트를 웹 화면에서 즉시 확인
- 결과 다운로드: 최종 결과 리포트를 PDF 파일 형태로 다운로드하는 기능
2-2. 결과 분석 및 리포트 생성: 클라이언트가 제공하는 엑셀 로직을 기반으로 시스템을 구축합니다.
- 데이터 처리: 사용자가 제출한 응답 데이터를 시스템에 저장 및 가공
- 통계 분석 자동화: 기존 엑셀 파일에 정의된 계산 공식 및 t-test 통계 처리 로직을 백엔드 시스템 구현
- 리포트 생성: 분석된 데이터를 바탕으로 상세 결과가 담긴 리포트 페이지 생성
2-3. 관리자 기능: 최소한의 서비스 모니터링을 위한 관리자 페이지를 구축합니다.
- 접속 로그 확인: 서비스 접속 현황을 파악할 수 있는 로그 데이터 조회 페이지
