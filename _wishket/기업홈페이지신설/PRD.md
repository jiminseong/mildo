# 기업 공식 홈페이지 및 관리자 구축 PRD

## 1. 문서 개요

- 프로젝트명: 다른화학(주) (Dareun Chemicals) 공식 홈페이지 신규 구축 데모
- 대상 폴더: _wishket/기업홈페이지신설
- 문서 목적: 공고 요구사항을 기능 단위로 구체화하고, 개발/디자인/운영 기준을 명확히 정의한다.
- 레퍼런스: www.hkchemicals.com (전반적인 레이아웃 및 톤앤매너 벤치마킹용이며, 컬러/무드는 다른화학 브랜드에 맞게 독자적인 톤으로 차별화)

## 2. 프로젝트 목표

- 기업(다른화학) 브랜드 신뢰도 및 친환경 기업 이미지 향상
- 방문자의 정보 탐색 시간 단축
- 제품 및 공지 운영의 내부 효율화
- 고객 문의 접수/관리 프로세스 일원화
- 모션 기반 인터랙션으로 브랜드 인상과 콘텐츠 몰입도 강화

## 3. 범위 정의

### 3.1 포함 범위 (1차 오픈)

- 반응형 기업 홈페이지 구축(PC/Tablet/Mobile)
- 회사 소개, 제품/서비스 소개, 공지/뉴스, 문의 기능
- 관리자 페이지(제품/공지 CRUD, 문의 조회)
- 기본 SEO 및 배포

### 3.2 협의 범위

- 문의 상태 관리 고도화(처리 상태, 담당자, 메모)
- 제품 카테고리 다단계 구조
- 파일 첨부/다운로드(브로슈어 등)

### 3.3 제외 범위 (2차 권장)

- 다국어
- CRM/ERP 연동
- 고급 통계 대시보드
- 마케팅 자동화

## 5. 상세 기능 요구사항

### FR-01 메인 페이지

- 기업 핵심 가치/미션 강조 섹션
- 주요 제품 하이라이트 섹션
- 공지/뉴스 최신 글 노출
- 문의 유도 CTA 배치

### FR-02 회사 소개 페이지

- 기업 비전/미션
- 연혁
- 조직/인사말(선택)
- 오시는 길/연락처

### FR-03 제품 및 서비스 페이지

- 카테고리 기반 목록
- 제품 상세(이미지, 설명, 주요 스펙)
- 관련 제품 추천
- 문의 유도 버튼

### FR-04 고객 문의

- 문의 폼(이름, 연락처, 이메일, 문의 내용)
- 개인정보 수집 동의 필수
- 제출 성공/실패 피드백
- DB 저장 및 관리자 조회 가능

### FR-05 공지/뉴스

- 리스트/상세 조회
- 게시글 등록/수정/삭제
- 게시 상태 관리(게시/비게시)

### FR-06 관리자 - 제품 관리

- 제품 등록/수정/삭제(CRUD)
- 카테고리 지정
- 노출 상태(게시/비게시)
- 썸네일/이미지 관리

### FR-07 관리자 - 문의 관리

- 문의 목록 조회
- 상세 조회
- 상태값 관리(신규/처리중/완료) (협의)
- 검색/필터(이름, 날짜, 상태)

### FR-08 공통

- 반응형 웹
- 기본 접근성 준수(label, 키보드 포커스)
- 주요 페이지 SEO 메타데이터

### FR-09 모션/애니메이션

- GSAP ScrollTrigger를 활용한 프리미엄 스크롤 기반 메인 히어로 시퀀스 구현 (배경 및 텍스트 교차 전환)
- 섹션 스크롤 진입 시 reveal 애니메이션 적용(상하 이동 + 페이드 및 stagger)
- 제품/뉴스 카드 hover 시 확대/그림자/포인트 라인 모션 적용
- 페이지 전환 시 과도하지 않은 부드러운 트랜지션 적용
- 문의 제출/관리자 저장 완료 시 상태 피드백 애니메이션 제공
- 사용자 환경설정이 reduce motion인 경우 최소 모션 모드로 자동 전환

## 7. 비기능 요구사항

- 성능: 핵심 페이지 LCP 기준 개선(이미지 최적화, 코드 분할)
- 품질: lint/build 통과
- 보안: 관리자 접근 보호(로그인), 입력값 검증
- 운영: 오류 로그 확인 가능한 최소 모니터링 체계
- 모션: 60fps 목표, 애니메이션 구간은 transform/opacity 중심 구현
- 접근성: prefers-reduced-motion 대응으로 모션 강도 제어

## 8. 수용 기준 (Acceptance Criteria)

- AC-01: 모바일/태블릿/PC에서 레이아웃이 깨지지 않는다.
- AC-02: 제품 등록/수정/삭제 후 사용자 페이지에 정상 반영된다.
- AC-03: 공지 등록/수정/삭제가 정상 동작한다.
- AC-04: 문의 폼 제출 데이터가 관리자에서 조회된다.
- AC-05: 주요 페이지 메타 타이틀/설명이 설정된다.
- AC-06: lint/build 에러 없이 배포 가능 상태다.
- AC-07: 메인/목록/상세 화면에 정의된 모션 시나리오가 적용된다.
- AC-08: reduce motion 환경에서 핵심 정보 전달에 문제 없이 동작한다.
