# 기업 홈페이지 메인페이지 리뉴얼 지원서

## 기본 입력값

- 지원 금액: 1,000,000원
- 지원 기간: 4일
- 관련 포트폴리오: 있습니다

### 금액/기간 제안 메모

- 현재 퍼블리싱 소스코드를 보유 중이시며, Figma 디자인에 맞춘 UI/인터랙션(모션) 고도화 및 영문 버전 추가를 요청하신 만큼 기간 내(4일)에 충분히 완수가 가능합니다.
- React 기반 기업 사이트 스크롤 인터랙션 개발에 대한 최근 경험(GSAP 등 활용)을 바탕으로, 퍼포먼스 이슈 없이 빠르고 깔끔하게 안정화할 수 있어 공고 예산 및 일정 그대로 제안드립니다.

## 제출용 본문

```text
지원 금액: 1,000,000원
지원 기간: 4일

<프로젝트 진행 제안>
- 현재 보유 중이신 1차 리액트(React) 퍼블리싱 소스코드를 빠르게 분석한 뒤, 멈춰 있는 디자인/인터랙션 구현을 끝까지 매듭짓는 작업으로 이해했습니다.
- 특히 메인 페이지는 '회사의 첫인상'이므로 디자인 파일(Figma) 상의 미세한 간격, 타이포그래피뿐만 아니라 스크롤 모션 및 브라우저 호환성까지 꼼꼼하게 다듬어야 합니다. 
- 이 프로젝트는 새로운 기능 개발보다, 기존 코드를 받아 빠르게 렌더링을 최적화하고 세련되게 "완성"해내는 속도와 디테일이 가장 중요할 것입니다.
- 영문 버전 추가 또한 기 구축된 리액트 라우팅 혹은 컴포넌트 구조에 맞춰 가장 관리하기 쉬운 형태로 세팅해 드리겠습니다. (이후 운영/배포는 클라이언트 측 진행이시므로, 깔끔한 코드를 산출물로 전달해 드립니다.)

[예상 일정]
- 1일 차: 전달 주시는 소스코드/디자인 리뷰 및 기존 코드의 누락분(컴포넌트/디자인 스펙) 보완
- 2일 차: 주요 인터랙션(모션/이펙트) 구현 및 PC/모바일 반응형 레이아웃 교정
- 3일 차: 영문 버전 메인 페이지 퍼블리싱 구현 및 줄바꿈/텍스트 최적화
- 4일 차: QA(크로스 브라우징, 버그 수정) 및 최종 소스코드 정리 후 인계

<유사 프로젝트 진행 경험>
- 바로 최근까지 B2B 기업 커스텀 홈페이지 데모를 React(Next.js) 기반으로 제작하며, GSAP(ScrollTrigger)를 활용해 방문자가 제품과 브랜드를 매력적으로 느끼는 '스크롤 체감형 모션'을 직접 구현한 바 있습니다.
- 반응형 웹 구조에서 흔히 발생하는 Sticky(고정) 버그나 레이아웃 깨짐 이슈들을 성공적으로 해결하고, 데스크탑과 모바일 모두 일관되게 프리미엄한 인상을 주는 인터랙션을 세팅하는 데 능숙합니다.
- 기존 다른 개발자의 코드를 이어받아 구조를 파악하고, 트러블슈팅하며 프로젝트를 무사히 완주한 유지보수/리뉴얼 경험 역시 풍부합니다. 지속적인 관계 유지를 원하시는 점 적극 환영합니다.

<사용 기술과 툴>
- Frontend: React.js, (필요에 따라 GSAP / Framer Motion 등으로 이펙트 적용)
- 작업 방식: 전달 주시는 코드 저장소(Git)를 통해 Branch 작업하거나 지정해 주시는 방법으로 진행 가능
- 커뮤니케이션: 위시켓, 화상 회의 등 즉각적인 피드백 반영
- 산출물: 완성된 React 퍼블리싱 (모션/다국어 적용 완료본) 소스코드

관련 포트폴리오:
- [D사(화학/정밀소재 B2B) 글로벌 웹사이트 리뉴얼]: 메인 첫 화면에 GSAP ScrollTrigger를 활용한 풀스크린 이미지 전환 및 텍스트 모션(Fade/Up)을 구현했습니다. 모바일 기기에서의 스크롤 고정(Sticky) 레이아웃 버그를 직접 트러블슈팅하고, 영/국문 버전을 분리된 컴포넌트 구조로 퍼블리싱하여 퍼포먼스 이슈 없이 납품을 완료했습니다.
- [MOUMOUTE]: 커머스 수준에 준하는 정밀 사용자 애니메이션 및 반응형 퍼블리싱 (부분 리뉴얼 및 코드 최적화 수행 완료)
- [제천남여헬스클럽]: 오프라인 업체의 온라인 거점 구축, 직관적 UI 설계 및 반응형 랜딩 페이지 퍼블리싱 완료
```

## 작성 메모

- 클라이언트가 이미 코드를 가지고 있고 "이펙트/모션/영문판"이 필요하다는 점을 공략했습니다.
- 예산이 크지 않은 대신, "본 작업 외에도 꾸준히 함께할 개발자"를 찾는다는 공고의 핵심 니즈에 맞추어, 코드를 깔끔히 이어받아 유지보수까지 든든하게 할 수 있다는 '안정감'을 크게 어필하는 방향으로 썼습니다.
- 최근에 작성된 다른화학(GSAP 모션 데모) 프론트엔드 작업 성과를 포트폴리오 1순위로 배치했습니다.
