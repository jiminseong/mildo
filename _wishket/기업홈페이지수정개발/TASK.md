# 기업 홈페이지 메인페이지 리뉴얼 퍼블리싱

## 개요
- **목적**: 기존 기결과물(1차 퍼블리싱 완료 상태)을 인수인계 받아, 피그마 디자인에 맞춘 메인페이지 리뉴얼 퍼블리싱 및 모션/이펙트 개발 완료. 영문 버전 추가.
- **예산**: 1,000,000원
- **기간**: 4일
- **산출물**: React 퍼블리싱 소스코드 (클라이언트 측에서 배포 진행)

## 기술 스택
- 프론트엔드: React (PC/MO 반응형)
- 모션/이펙트: CSS Animation, Framer Motion 또는 GSAP (선택)

## 핵심 과제 (Task)

### 1. 코드 인수인계 및 현황 파악 (Day 1)
- 클라이언트로부터 전달받은 1차 퍼블리싱 소스코드 구조 및 상태 리뷰
- Figma 디자인 파일 분석 및 미구현/수정 필요 영역 도출
- 컴포넌트 구조 재정비 (필요 시)

### 2. 메인페이지 리뉴얼 및 반응형 최적화 (Day 1 - 2)
- Figma 디자인 스펙에 맞게 UI 개선 (Typography, Spacing, Color 등 정교화)
- PC 및 모바일(MO) 환경에서의 반응형 레이아웃 완벽 대응
- 누락된 컴포넌트 퍼블리싱

### 3. ಇ펙트 및 모션 개발 (Day 2 - 3)
- 메인페이지 진입 시 스크롤 애니메이션 및 전환 효과 구현
- 버튼 Hover, 스크롤 기반 등장(Reveal) 모션 등 디테일한 인터랙션 추가
- (최근 GSAP를 활용한 기업 홈페이지 데모 개발 경험 적극 활용)

### 4. 영문 버전 구현 (Day 3)
- 클라이언트가 제공하는 영문 번역본(Figma)을 바탕으로 영문 버전 메인페이지 퍼블리싱 추가
- i18n 라이브러리 도입 혹은 페이지 라우팅 분기 처리 (기존 코드 구조에 맞춰 협의 후 결정)
- 영문 텍스트 길이에 따른 레이아웃(줄바꿈 등) 최적화

### 5. 안정화 및 산출물 전달 (Day 4)
- 크로스 브라우징 및 디바이스 테스트 (QA)
- 애니메이션 성능 및 버그 확인
- 최종 소스코드 패키징 및 클라이언트에 전달 (배포 지원 불필요)
- 향후 지속적인 유지보수/추가 개발을 대비한 코드 클린업 및 주석 작성
